# Admin MVP Flows

## Manual Pricing

- View current provider/API price.
- Enter manual price with reason and validity.
- Confirm sensitive action.
- Verify audit trail.

## Market Status

- View current status per market.
- Switch open/closed/manual-only when authorized.
- Confirm reason.
- Verify affected customer flows show correct state.

## Approvals

- Review manual trade queue.
- Review withdrawal queue.
- Review delivery queue.
- Approve/reject with reason.
- Verify audit trail.

## Reconciliation

- View provider mismatch report.
- View bank/payment mismatch report.
- View wallet/ledger mismatch report.
- Mark investigation states only through backend-approved actions.
