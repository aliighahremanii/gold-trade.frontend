# Shared UI primitives

Accessible, token-driven, highly customizable presentation components for the trading platform.

**Implementation prompt (optional):** `docs/prompts/03-shared-ui.md`  
**Review prompt:** `docs/review/prompts/03-shared-ui-review.md`  
**Design notes:** `docs/design-system.md`

Includes generic primitives (buttons, inputs, dialogs, tables) and financial display shells (amounts, status badges, quote countdown, confirmation patterns).

Module-specific workflow UI belongs under `src/modules/{module}/components`, not here.
